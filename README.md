# PlayerSettings-By-Blockem13
A customizable player profile and settings system with bios, languages, and social links.
# PlayerSettings - Dein persönliches Profil-System!

**Tested Major MC Versions:**

* 1.21

**Languages Supported:**

* English

## Features:

### /settings - Persönliches Einstellungsmenü

* Eigenen Skin-Kopf mit Namen
* Bio mit Namensschild setzen
* Sprachauswahl (Deutsch/Englisch/Französisch)
* Social Media Links verwalten

### /profile [Spieler] - Profile anderer Spieler anzeigen

* Skin-Kopf mit UUID
* Gesetzte Bio
* Ausgewählte Sprache
* Verlinkte Social Media Accounts (Twitch, Twitter, YouTube, bald auch Federtweet)

### /bio <Text> - Schnell die eigene Bio ändern

### mcheads.ru Integration

* Authentische Player-Köpfe mit IDs
* Language-Kopf (Google Chrome)
* Länder-Köpfe mit korrekten Flags
* Social Media Icons

### Vollständige Datenspeicherung

* YAML-basierte Spielerdaten
* Persistente Speicherung
* Einfach erweiterbar

## Perfekt für:

* RPG-Server
* Towny-Server
* Lobby-Systeme
* Netzwerke

## Kompatibilität

* Kompatibel mit Paper 1.21+
* Einfach installieren und loslegen!

*(Wird auf Blockmc.de benutzt)*
